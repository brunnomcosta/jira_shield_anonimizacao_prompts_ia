Templates de prompt

O template padrao fica em chrome-extension/prompt-templates.js. Aqui voce pode sobrescrever a base, adicionar regras complementares e revisar a versao consolidada do prompt enviado a LLM.

Esta configuracao interfere apenas no prompt enviado a LLM. As regras internas de anonimização continuam fora deste fluxo e permanecem centralizadas no pipeline de sanitizacao.
Precedencia: override do usuario > template padrao do arquivo. Depois disso, as regras complementares do usuario sao anexadas ao final.
O plugin deve conseguir enxergar qual arquivo fisico define o template padrao desta opcao.
Estas sao as regras padrao mantidas no arquivo fisico da extensao. Elas nao alteram a anonimização e servem como base para o prompt enviado a LLM.
Use apenas se voce quiser substituir totalmente a base padrao para este tipo de prompt.
Estas instrucoes sao anexadas ao final da base consolidada.
Os placeholders da issue sao preservados neste preview e sao resolvidos apenas na hora de gerar o prompt.
