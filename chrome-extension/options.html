SHIELD

Configuração da extensão Chrome para exportação LGPD do Jira.

Options

Resumo operacional

A extensão cobre o módulo de exportação e gera os arquivos de entrada do diagnostics.js.

PDF anonimizado metadata JSON proxy Jira API Zendesk opcional histórico local

Jira

Obrigatório. Sem JIRA_BASE_URL a extensão não consegue ler a issue.

Sem barra final. Ex.: https://jiraproducao.totvs.com.br
Campo Jira que guarda o ticket Zendesk vinculado.

Autenticação Jira

Prioridade: token pessoal, depois usuário/senha, depois sessão já aberta no navegador.

Recomendado para Jira 8.14+.
Subpasta relativa dentro de Downloads.

Zendesk

Opcional. Se não preencher, a extensão tenta o proxy Jira e depois a aba Zendesk da issue.

O diagnóstico com LLM, leitura de workspace local e análise ERP/mobile continuam no CLI Node.js.

IA para geração de documentação

Após exportar uma issue com ticket Zendesk vinculado, um botão abrirá a IA escolhida para gerar um Documento Técnico TOTVS (DT).

IA preferida
